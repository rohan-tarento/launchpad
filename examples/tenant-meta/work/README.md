# Work manifests

Generated by `/generate-work-manifest` after PRD sign-off. One file per initiative: `work/INIT-<id>.yaml`.

```bash
launchpad seed-work --config work/INIT-<id>.yaml --dry-run
launchpad seed-work --config work/INIT-<id>.yaml --apply
```
